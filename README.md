# wright
An AI-assisted IELTS writing tool built with vibe coding.


Wright is an AI-assisted IELTS writing tool built through vibe coding. It helps users practice IELTS writing, receive AI feedback, generate model essays, review past work, organize key learning points, and turn them into targeted exercises.


Wright 是一个通过vibe coding 构建的AI辅助雅思写作工具，能够帮助用户进行雅思写作练习，获得AI批改、AI范文，复盘过往记录，整理知识点并进行针对性练习。
